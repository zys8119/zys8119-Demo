### 9. 任务执行的洋葱模型

```typescript
const t = TaskPro()
t.addTask(async (next)=>{
    console.log(1, 'start')
    await next()
    console.log(1, 'end')
})
t.addTask(async (next)=>{
    console.log(2)
})
t.addTask(async (next)=>{
    console.log(3)
})
t.run(); // 1 start, 2, 3, 1 end
```
